# FunRun乐跑

## Introduction

FunRun乐跑是一款基于微信小程序的跑步类应用，用户可以通过该应用记录自己的跑步数据，查看自己的跑步历史记录、跑步排名、跑步成就、跑步数据分析等专业数据，帮助用户更好的了解自己的跑步情况，从而更好的进行跑步训练。

## Features

- [x] 用户登录
- [x] 用户注册
- [x] 用户信息修改
- [x] 用户信息查看
- [x] 跑步数据记录
- [x] 跑步数据查看
- [x] 跑步数据分析
- [x] 跑步排名
- [x] 跑步成就
- [x] 跑步计划
- [x] 跑步打卡
- [x] 跑步分享

## Requirements

- 微信小程序

## Usage

- 微信小程序搜索“FunRun乐跑”即可使用
